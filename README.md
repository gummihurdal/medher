# Með þér

A small private love letter, set to music.

Drop image files into `photos/`, then list them in `photos.json`:

```json
["01.jpg", "02.jpg", "03.jpg"]
```

The page reads them in order and crossfades through them as the song plays.

Song: *Með þér* — Ragnheiður Gröndal & Linda Sigurðardóttir.
